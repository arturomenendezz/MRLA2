COMO FUNCIONA UN DISCO DURO MÁGNETICO

Para empezar, un disco duro es un dispositivo de almacenamiento de datos no volátil que emplea un sistema de grabación magnética para almacenar datos digitales de forma rápida y segura. Para almacenar datos se utilizan únicamente dos valores representados por 0 y 1 (también denominado código binario).
